## 风险提示

1. **F-001 + F-002 强耦合**: 两者都涉及 MATLAB `main_multi.m`，但修改不同段落（方向透传 vs 静态轨迹），可独立 commit
2. **F-004 调查周期不确定**: 干扰不可观测的根因可能需要端到端 debug，实际工时可能超预估
3. **F-003 依赖 F-001**: F-001 修好后 NMSE 可能自动改善（如果根因是方向错误导致 ideal/estimated 比较错位），也可能仍需独立修轴顺序问题
4. **F-008 改动面大**: 如果决定本轮修，需要修改 `_generate_one_sample()` 的核心信道生成逻辑，波及面广
