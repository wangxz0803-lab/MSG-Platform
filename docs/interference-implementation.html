2. 处理流水线（6 步）

1 多小区信道生成 — 全 K 小区 → 单 UE
对每个 UE，生成从 所有 K 个小区到该 UE 的信道矩阵。每个信道包含小尺度衰落（CDL/TDL/QuaDRiGa/RT）。
输出张量：[K, T, RB, BS_ant, UE_ant]
2 路径损耗 + 相对功率缩放
计算每个小区到 UE 的路径损耗 PL(k)，得到接收功率 Rx(k) = Tx - PL(k)。
服务小区信道保持单位功率，干扰小区按 √(P_k / P_serving) 幅度缩放，保持物理一致的相对强度。
3 服务小区选择 — 最强 RSRP
serving_idx = argmax(rx_power_dbm)
选择接收功率最大的小区作为服务小区。其余 K-1 个为干扰小区。
4 提取 h_interferers + interference_signal
从全 K 小区信道中提取非服务小区：
h_interferers = stack(h_all[k] for k ≠ serving) → [K-1, T, RB, BS, UE]
聚合干扰信号（沿干扰小区和 BS 天线维度求和）：
interference_signal = Σ_cells Σ_bs h_interferers → [T, RB, UE]
5 SNR / SIR / SINR 计算
SNR= P_serving - N₀ (dBm 域相减)
SIR= P_serving / Σ P_interferer (线性域)
SINR= 1 / (10-SNR/10 + 10-SIR/10) (倒数合成)
其中 N₀ = -174 + NF + 10·log₁₀(BW) dBm
6 干扰注入到 PHY 流水线
UL SRS 接收：Y = H_serving·X_serving + Σ_k H_k·X_k + noise → LS/MMSE 信道估计
DL CSI-RS 接收：Y = H_serving·X_serving + Σ_j H_j·X_j + noise → LS/MMSE 信道估计
干扰导频来自不同小区的 Gold/ZC 序列（不同 c_init / root），天然正交性不完美 → 估计误差

5. UL vs DL 干扰差异

DL 下行干扰（CSI-RS）

  • 干扰来源：邻小区 BS 发送的 CSI-RS 导频
  • 每个小区使用不同 c_init 的 Gold 序列 → QPSK 导频
  • 接收信号：Y = H_s·X_s + Σⱼ Hⱼ·Xⱼ + n
  • 干扰功率取决于邻小区路径损耗差（宏分集增益）
  • UE 通过 LS/MMSE 估计 H_serving，干扰作为额外噪声
% dl_csirs_pipeline.m
for k = 1:K
  c_init_k = mod(c_init + k, 2^31-1);
  X_all(k,:) = qpsk_from_bits(
    gold_prbs(c_init_k, 2*N_RB));
end
% 接收 = 服务信号 + 干扰 + 噪声

UL 上行干扰（SRS）

  • 干扰来源：邻小区 UE 发送的 SRS 导频
  • 每个小区使用不同 root 的 ZC 序列
  • 接收信号：Y = H_s·X_s + Σₖ Σᵤ Hₖᵤ·Xₖᵤ + n
  • 需要生成干扰 UE（num_interfering_ues 个/邻小区）
  • 每个干扰 UE 独立路径损耗、独立小尺度衰落
// internal_sim.py
for k in interferer_cells:
  intf_ues = place_ues(N_intf, ...)
  for n in range(N_intf):
    h_ue = _generate_cdl_channel(
      tx=UE_panel, rx=BS_panel)
    h_ue *= sqrt(P_n / P_serving)

6. SRS/CSI-RS 流水线中的干扰注入

UL SRS 信道估计流水线

干扰如何影响估计精度：每个小区的 SRS 序列虽然根序列不同（近似正交），但在有限带宽上互相关不为零。 干扰功率越大 → LS 估计中的等效噪底越高 → pre-SINR 降低。MMSE 平滑可部分抑制干扰带来的频域噪声。 
// 简化的 UL SRS 接收（所有源逻辑一致）

// 1. 各小区发送 SRS
for k = 1:K
    X_srs(k, :) = zc_sequence(root_k, N_RB);    // ZC 序列，root 因小区而异
end

// 2. BS 接收 = 服务 UE + 所有干扰 UE + 噪声
Y = H_serving .* X_srs(sid,:)                  // 服务信号
  + Σ_k≠sid (H_k .* X_srs(k,:))              // 邻小区干扰
  + noise;                                    // AWGN

// 3. LS 估计（仅在 SRS 导频 RB 上）
pilot_rbs = srs_rb_indices(srs_cfg, slot);    // 可能是跳频子集
H_ls = Y(pilot_rbs) .* conj(X_s) ./ |X_s|²;  // 干扰残留在 H_ls 中

// 4. 插值到全带宽
H_est = interp_to_full_band(H_ls, pilot_rbs, N_RB);

// 5. 可选 MMSE 平滑（抑制干扰 + 噪声）
if mode == 'ls_mmse'
    W = R_hh / (R_hh + (1/SNR) * I);          // Wiener 滤波
    H_est = W * H_est;
end

// 6. Pre-SINR（衡量估计质量）
pre_sinr_rb = |H_true|² ./ |H_true - H_est|²;  // 逐 RB

DL CSI-RS 信道估计流水线

// 简化的 DL CSI-RS 接收

// 1. 各小区发送 CSI-RS（Gold 序列 → QPSK）
for k = 1:K
    c_init_k = mod(c_init + k, 2^31 - 1);
    X_csi(k, :) = gold_qpsk(c_init_k, N_RB);
end

// 2. UE 接收 = 服务 BS + 所有邻 BS + 噪声
Y = Σ_bs H_serving(bs,:) .* X_csi(sid,:)       // 服务信号（所有天线相干）
  + Σ_k≠sid Σ_bs H_k(bs,:) .* X_csi(k,:)    // 邻小区干扰
  + noise;

// 3. LS 估计（全频带，density=1）
H_ls_ue = Y .* conj(X_s) ./ |X_s|²;

// 4. 分解到 BS 天线维（利用真实信道比例）
ratio(bs) = H_true(bs,:,:) ./ Σ_bs H_true(bs,:,:);
H_est(:,:,bs,:) = H_ls_ue .* ratio(bs);

// 5. SINR = 服务功率 / (干扰功率 + 噪声)
sinr = sig_pow / (intf_pow + noise_linear);

7. 核心公式

指标公式说明
噪声功率 N₀ -174 + NF + 10·log₁₀(12·SCS·N_RB) dBm NF 默认 7 dB，可配置
接收功率 Rx(k) Tx_power - PL(k) dBm PL 含阴影衰落（log-normal）
相对幅度 √(10^(Rx_k/10) / 10^(Rx_s/10)) 干扰信道 × 此系数
SNR Rx_serving - N₀ dB 无干扰场景
SIR P_s / Σ_{k≠s} P_k (线性) → dB 纯干扰比
SINR -10·log₁₀(10^(-SNR/10) + 10^(-SIR/10)) 倒数合成
Pre-SINR (per RB) ‖H_true‖² / ‖H_true - H_est‖² 估计质量指标
SSB RSRQ N_RB · RSRP_s / Σ_k RSRP_k 参考信号接收质量
